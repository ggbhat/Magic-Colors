Magic Color

The main goal of this puzle game is to make two boards same. There are two boards one is Master board and Goal Board. User need to align the Master Board Cells same as Goal Board. The white colored cell is empty. User can start playing by moving colored cell to white cell. Cells moved only to its adjusent cell.
To win the game you should arrange the master board so that it matches the Goal board.
